## Installation
If no python env yet:
```
python3 -m venv venv
source venv/bin/activate
```

Subsequent calls:
```
source venv/bin/activate

# Windows
.\venv\bin\Activate.ps1

# run command...
```

Install dependencies:
```
python3 -m pip install -r requirements.txt
```

## Building
```
make html

# Windows
 .\make.bat html
```
