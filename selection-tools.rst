Selection Tools
=========================================

.. image:: _static/images/manual/selection-tools.png

.. tip::
    With all selection tools (except ``Select Linked by Material``), while holding the left mouse button and dragging in the viewport, hold :kbd:`Shift` to add to the selection or hold :kbd:`Alt` to remove/subtract from the selection.

1. Single Selection: select a single vertex per click (i.e. precision selection).
2. Lasso Selection (:kbd:`L`)
3. Rectangle Selection
4. Ellipse Selection

5. Select Linked by Material: choose one of the mesh's materials to select all vertices of the faces assigned to that material (just like Blender's "Select Linked" feature).

    - This is only useful when the mesh has multiple materials.
    .. image:: _static/images/manual/select-linked-by-material.gif

6. Select All
7. Deselect / Clear Selection (:kbd:`Shift+L`)

8. Invert Selection

    .. image:: _static/images/manual/select-invert-selection.gif
